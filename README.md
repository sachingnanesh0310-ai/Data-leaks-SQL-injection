# SQL Injection Detection System

A simple cybersecurity project built using HTML, CSS, and JavaScript that detects possible SQL Injection attempts in user input fields.

This project is designed for educational and defensive security purposes only.

---

## 📌 Features

- Detects common SQL Injection patterns
- Real-time input validation
- Modern responsive UI
- Beginner-friendly source code
- Pure HTML, CSS, and JavaScript
- Lightweight and easy to run

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
